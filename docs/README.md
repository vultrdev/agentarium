## Canonical public docs

The public documentation home is:
- `https://github.com/vultrdev/agentarium-docs`

Use that repo as the public source of truth for:
- product docs
- architecture docs
- contracts
- governance
- contribution surfaces
- media/demo planning

This local directory can exist for implementation convenience, but public documentation quality should be maintained in the docs repo as well.
